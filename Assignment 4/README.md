Vector Operations Program
Description

This Java program performs basic vector operations for 2D and 3D vectors:

Addition

Subtraction

Dot Product

It uses exception handling to check for invalid vector dimensions and dimension mismatch.

Files

Vector.java → Contains vector logic and operations

VectorException.java → Custom exception class

VectorOperations.java → Main program

Concepts Used

Classes and Objects

Constructors

Arrays

Methods

Exception Handling

How to Run

Compile all files:

javac *.java

Run the program:

java VectorOperations
Functionality

The program:

Accepts vector dimension (2 or 3)

Reads vector components

Performs operations

Displays results

Handles errors using exceptions

Output

Shows:

Vector Addition

Vector Subtraction

Dot Product

Result: Program executed successfully with correct vector operations.
